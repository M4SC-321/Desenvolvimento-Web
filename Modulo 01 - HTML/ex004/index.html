Nesse texto tem um termo em negrito usando a tag <b> (sem semântica).

Nesse texto tem um termo em destaque usando a tag <strong> (semântica).

Itálico/Ênfase

Nesse texto tem um termo em itálico usando a tag <i> (sem semântica).

Nesse texto tem um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Texto Grande e Pequeno

Nesse texto tem um termo grande e um termo pequeno, usando respectivamente as tags <big> (semântica, mas em desuso) e <small> (semântica).

Texto Excluído

Podemos marcar um termo/texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del> (semântica).

Texto Inserido

Podemos marcar um termo/texto como inserido para dar ênfase e indicar que o termo/texto foi adicionado posteriormente, usando a tag <ins> (semântica).

Existe também o subilinhado com a tag <u> (sem semântica).

Texto Sobrescrito

Nesse texto tem um termo sobrescrito usando a tag <sup> (semântica).

Nesse texto tem um termo subscrito usando a tag <sub> (semântica).
